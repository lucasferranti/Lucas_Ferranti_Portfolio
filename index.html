Projet M2 — Injection de fautes sur CNN LeNet-5 quantifié INT8
Objectif : mesurer l’impact de bit-flips sur un réseau (MNIST) via campagnes massives d’injection de fautes.
  • Développé un framework d’injection de fautes en C (instrumentation via macro INLINE_BITFLIP)
  • Automatisé les campagnes via scripts Python et génération de résultats CSV
  • Exécuté > 3 millions d’inférences en environnement C instrumenté
  • Analysé statistiquement les erreurs par couche / bit / direction (MSB plus sensibles, couches C1 & C3 vulnérables)

Projet Robotique — Contrôle de trajectoire & asservissement sur FPGA (ZYNQ7)
Objectif : système de contrôle temps réel intégrant PWM, mesure vitesse, obstacles, suivi de ligne, UART.
  • Développé un générateur PWM (125 MHz → 20 kHz) et une FSM VHDL (trajectoire carrée)
  • Validé le design via simulations temporelles (chronogrammes, transitions d’états)
  • Mesuré la vitesse via encodeur incrémental (détection fronts + compteur matériel)
  • Transmis les données via UART (ASCII) et géré capteurs/interruptions via AXI GPIO
  • Ajouté détection obstacle (IR + ADC) et logique d’évitement, validation sur robot réel
